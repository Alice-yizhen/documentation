### 概述

[Punica Suite](https://punica.ont.io) 是基于Ontology的第一个dApp开发框架， 为开发者提供了一整套开发工具：

* [Punica Python CLI](https://github.com/punicasuite/punica-python) 和 [Punica TypeScript CLI](https://github.com/punicasuite/punica-ts) - 是命令行工具，支持编译、部署、调用、测试智能合约
* [Punica boxes](https://punica.ont.io/boxes/) - 提供多款dApp模板和案例
* [Solo-chain](https://github.com/punicasuite/solo-chain/releases) - 本体的一键式（私有）区块链。 

Punica的目标是提供开发者完善的工具集和资源，方便开发者快速开发dApp。

优势：
1. Punica是本体第一套dapp开发框架，帮用户节省至少50%开发时间。
2. 提供大量教材文档和视频，方便初学者入门。
3. 开发了智能合约测试框架，支持单元测试和功能测试，比使用SDK测试门槛降低许多。
4. 智能合约编译部署测试为一体，节省开发时间。smartx中已集成debug功能，后续支持命令行debug功能。
5. solo-chain方便用户实时查看链上数据，比在测试网或搭建私网方便50%以上。
6. 提供多种SDKs和dApi的案例，适合不同用户群开发dApp。
